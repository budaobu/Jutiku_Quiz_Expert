# Quiz Markdown Specification v2.0

为了确保题目在 Markdown 环境下的可读性、解析一致性与多模态资源的正确引用，请遵循以下排版协议。

---

## 0. 题库文档结构

每个题库 Markdown 文档应包含 文档标题、元数据区块、题目列表 三个部分。

### 完整文档模板
```markdown
# 题库 - {YYYY-MM-DD_HHMMSS}

## 📋 元数据
- 来源文档: 文件名.docx
- 来源哈希: sha256:a1b2c3d4...
- 生成时间: YYYY-MM-DD HH:MM:SS
- 题目数量: 25
- 多模态题目: 8
- 生成器版本: 2.0.0

---

### [题目 ID] 题目标题
> 题干内容...

（题目详细内容）

---

### [题目 ID] 题目标题
> 题干内容...

（题目详细内容）

---
```

### 字段说明

| 字段 | 必需性 | 说明 |
|:---|:---|:---|
| 来源文档 | 必需 | 源文档的文件名，如 `高中数学必修2.docx` |
| 来源哈希 | 推荐 | 源文档的 SHA256 哈希，用于版本追溯和完整性验证 |
| 生成时间 | 必需 | 题库生成的时间戳，格式 `YYYY-MM-DD HH:MM:SS` |
| 题目数量 | 必需 | 文档中包含的题目总数 |
| 多模态题目 | 推荐 | 包含图片/音频/视频等资源的题目数量 |
| 生成器版本 | 推荐 | AI 出题系统的版本号，便于追踪生成方式 |

## 1. 题目基本结构

每一道题目应包含 题目标题、题干、选项列表、元数据表格、详细解析 五个部分。多模态题目需额外包含 资源引用 部分。

### 基础题目模板
```markdown
### [题目 ID] 题目标题
> 此处填写题干内容。如涉及数学公式，请使用 $LaTeX$ 渲染。

* A. 选项内容
* B. 选项内容
* C. 选项内容
* D. 选项内容

| 属性 | 内容 |
| :--- | :--- |
| 题型 | 单选题 / 多选题 / 判断题 / 计算题 / 证明题 |
| 难度 | 1 (基础) / 2 (进阶) / 3 (专家) |
| 分值 | 数字 (如: 5) |
| 知识点 | `标签1`, `标签2` |
| 正确答案 | [A] / [A, C] / 表达式 |

#### 💡 选项解析
* A. 针对 A 选项的逻辑说明及正误判定。
* B. 针对 B 选项的逻辑说明及正误判定。
* C. 针对 C 选项的逻辑说明及正误判定。
* D. 针对 D 选项的逻辑说明及正误判定。
```

## 2. 多模态资源引用规范

### 2.1 图片引用格式

标准语法：
```markdown
![语义标注](./assets/{SHA256}.{ext})
```

约束规则：
- 路径必须是相对路径，格式为 `./assets/{hash}.{ext}`
- 语义标注（alt text）必须描述图片的关键信息，便于无障碍阅读
- 哈希值使用完整 SHA256（64 位十六进制字符）

示例：
```markdown
![直角三角形ABC，已知∠C=90°，AC=3，BC=4](./assets/a3f7e2b1c4d5e6f7890abcdef1234567.png)
```

### 2.2 图片说明块

在图片引用后，使用引用块提供详细说明：
```markdown
> 图片说明：正方体 ABCD-A'B'C'D' 的三维结构图，边长为 2，标注了对角线 AC' 和向量 BA'。  
> 来源上下文：第 12 页，第 5 段 | 相关文本："空间向量的夹角定义为..."
```

### 2.3 资源哈希记录

在元数据表格中添加 `资源哈希` 字段：

| 属性 | 内容 |
| :--- | :--- |
| 资源哈希 | `sha256:a3f7e2b1c4d5e6f7890abcdef1234567` |

用途：用于验证资源完整性，支持自动化工具检查文件是否缺失或损坏。

## 3. 题型特殊规范

### 3.1 判断题

选项固定格式：
```markdown
* A. 正确
* B. 错误
```

正确答案格式：`[A]` 或 `[B]`

示例：
```markdown
### logic_005 关于逆否命题的判断
> 若 $p \rightarrow q$ 为真，则 $\neg q \rightarrow \neg p$ 必为真。

* A. 正确
* B. 错误

| 属性 | 内容 |
| :--- | :--- |
| 题型 | 判断题 |
| 难度 | 1 |
| 分值 | 3 |
| 知识点 | `命题逻辑` |
| 正确答案 | [A] |

#### 💡 选项解析
* A. 正确。这是逆否命题的等价性定理。
* B. 错误。逆否命题与原命题逻辑等价。
```

### 3.2 多选题

正确答案格式：`[A, C, D]`（包含所有正确选项）

解析要求：必须逐一分析每个选项的正误原因。

示例：
```markdown
### chem_008 化学平衡影响因素
> 对于放热反应 $N_2 + 3H_2 \rightleftharpoons 2NH_3$，下列措施能提高氨气产率的是？

* A. 增大压强
* B. 降低温度
* C. 使用催化剂
* D. 移除生成的氨气

| 属性 | 内容 |
| :--- | :--- |
| 题型 | 多选题 |
| 难度 | 2 |
| 分值 | 6 |
| 知识点 | `化学平衡`, `勒夏特列原理` |
| 正确答案 | [A, B, D] |

#### 💡 选项解析
* A. 正确。增压使平衡向体积减小方向移动，氨气产率提高。
* B. 正确。降温利于放热反应正向进行。
* C. 错误。催化剂只改变反应速率，不影响平衡位置。
* D. 正确。移除产物促使平衡正向移动。
```

### 3.3 计算题

选项字段：留空或省略（无选项）

正确答案格式：`表达式或数值`（如 `$a = 5m/s^2$，方向沿斜面向上`）

解析要求：必须包含完整的计算步骤。

示例：
```markdown
### phys_012 斜面物块受力分析

#### 📊 依赖资源
![斜面物块受力示意图](./assets/f8e3b2a1d5c6e7f8a9b0c1d2e3f4a5b6.png)
> 图片说明：质量 $m=2kg$ 的物块在倾角 $\theta=30°$ 的斜面上，受水平推力 $F=20N$ 作用。  
> 来源上下文：第 15 页，第 7 段 | 相关文本："斜面问题的关键在于正确分解受力..."

> 如图所示，质量 $m=2kg$ 的物块在倾角 $\theta=30°$ 的斜面上，受水平推力 $F=20N$ 作用。求物块的加速度。（取 $g=10m/s^2$）

| 属性 | 内容 |
| :--- | :--- |
| 题型 | 计算题 |
| 难度 | 3 |
| 分值 | 10 |
| 知识点 | `牛顿第二定律`, `受力分析` |
| 正确答案 | $a \approx 3.66m/s^2$，方向沿斜面向上 |
| 资源哈希 | `sha256:f8e3b2a1d5c6e7f8a9b0c1d2e3f4a5b6` |

#### 💡 解题步骤
1. 建立坐标系，分解受力：$F_x = F\cos\theta$，$F_y = F\sin\theta$
2. 沿斜面方向合力：
   $$F_{合} = F_x - mg\sin\theta = 20 \times \frac{\sqrt{3}}{2} - 2 \times 10 \times 0.5 = 10\sqrt{3} - 10$$
3. 计算结果：
   $$F_{合} \approx 17.32 - 10 = 7.32N$$
4. 由 $F=ma$ 得：
   $$a = \frac{F_{合}}{m} = \frac{7.32}{2} \approx 3.66m/s^2$$
```

### 3.4 证明题

选项字段：留空或省略

正确答案格式：`证明过程或结论`

解析要求：提供标准证明步骤。

示例：
```markdown
### math_015 三角形全等证明
> 已知：在 $\triangle ABC$ 和 $\triangle DEF$ 中，$AB = DE$，$\angle A = \angle D$，$AC = DF$。  
> 求证：$\triangle ABC \cong \triangle DEF$。

| 属性 | 内容 |
| :--- | :--- |
| 题型 | 证明题 |
| 难度 | 2 |
| 分值 | 8 |
| 知识点 | `三角形全等`, `SAS判定` |
| 正确答案 | 见解析 |

#### 💡 证明过程
证明：
1. 在 $\triangle ABC$ 和 $\triangle DEF$ 中：
   - $AB = DE$（已知）
   - $\angle A = \angle D$（已知）
   - $AC = DF$（已知）
2. 根据 SAS（边角边）判定定理：
   $$\triangle ABC \cong \triangle DEF$$

结论：得证。
```

## 4. 公式渲染规范

### 4.1 LaTeX 语法

行内公式：使用单个 `$` 包裹
```markdown
勾股定理：$a^2 + b^2 = c^2$
```

独占行公式：使用双 `$$` 包裹
```markdown
$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

### 4.2 常用符号规范

| 数学概念 | LaTeX 代码 | 渲染效果 |
|:---|:---|:---|
| 分数 | `\frac{a}{b}` | $\frac{a}{b}$ |
| 根号 | `\sqrt{x}` | $\sqrt{x}$ |
| 上标/下标 | `x^2`, `x_i` | $x^2$, $x_i$ |
| 求和 | `\sum_{i=1}^{n}` | $\sum_{i=1}^{n}$ |
| 积分 | `\int_{a}^{b}` | $\int_{a}^{b}$ |
| 希腊字母 | `\alpha`, `\beta`, `\theta` | $\alpha$, $\beta$, $\theta$ |
| 向量 | `\vec{v}` | $\vec{v}$ |
| 矩阵 | `\begin{pmatrix} a & b \\ c & d \end{pmatrix}` | $\begin{pmatrix} a & b \\ c & d \end{pmatrix}$ |

### 4.3 化学公式

使用 `\ce{}` 命令（需 mhchem 扩展）或纯文本：
```markdown
$\ce{H2O}$，$\ce{CO2}$，$\ce{N2 + 3H2 <=> 2NH3}$
```

## 6. 编写细则与禁止事项

### 6.1 强制要求

1. 文档必须包含元数据区块：标题下方紧跟 `## 📋 元数据` 部分
2. 判断题选项固定：`A. 正确`，`B. 错误`（不允许其他表述）
3. 多选题答案完整：正确答案必须列出所有字母，如 `[A, C, D]`
4. 公式必须使用 LaTeX：禁止使用纯文本表示数学表达式（如 `x^2` 必须写作 `$x^2$`）
5. 图片引用必须相对路径：禁止使用绝对路径或外部 URL
6. 每题必须包含解析：即使是判断题也要说明理由
7. 题目之间使用分隔线：每道题目后添加 `---`

### 6.2 禁止事项

| 禁止内容 | 原因 | 正确做法 |
|:---|:---|:---|
| "以上都对"/"以上都不对" | 缺乏教育价值，无法考察具体知识点 | 设计具体的逻辑干扰项 |
| 红-紫-蓝渐变色（UI 题目） | 用户偏好约束 | 使用其他配色方案 |
| 图片使用绝对路径 | 破坏可移植性 | 使用 `./assets/{hash}.ext` |
| 公式使用纯文本 | 渲染不一致 | 使用 LaTeX 语法 |
| 缺少资源哈希（多模态题） | 无法验证资源完整性 | 在元数据表格中添加 `资源哈希` 字段 |
| 元数据区块缺失 | 无法追溯题库来源 | 在文档开头添加完整元数据 |

### 6.3 推荐实践

- 选项干扰性：错误选项应基于常见误解或计算错误设计
- 解析深度：不仅指出正误，还要说明原因和推导过程
- 知识点标签：使用标准化术语，便于题库检索
- 难度分级：基于认知层次（记忆→理解→应用→分析）

### math_001 直角三角形边长计算

#### 📊 依赖资源
![直角三角形ABC，已知∠C=90°，AC=3，BC=4](./assets/a3f7e2b1c4d5e6f7890abcdef1234567.png)
> 图片说明：直角三角形 ABC 的示意图，标注了直角标记和已知边长。  
> 来源上下文：第 8 页，第 3 段 | 相关文本："勾股定理是平面几何中的基本定理..."

> 如图所示，在 $\triangle ABC$ 中，$\angle C = 90°$，$AC = 3$，$BC = 4$，求 $AB$ 的长度。

* A. 5
* B. 6
* C. 7
* D. $\sqrt{7}$

| 属性 | 内容 |
| :--- | :--- |
| 题型 | 单选题 |
| 难度 | 2 |
| 分值 | 5 |
| 知识点 | `三角函数`, `勾股定理` |
| 正确答案 | [A] |
| 资源哈希 | `sha256:a3f7e2b1c4d5e6f7890abcdef1234567` |

#### 💡 选项解析
* A. 正确。根据勾股定理：
  $$AB = \sqrt{AC^2 + BC^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$
* B. 错误。未正确应用勾股定理，可能是直接相加 $AC + BC - 1 = 6$ 的错误思路。
* C. 错误。计算错误，可能混淆了加法与平方和。
* D. 错误。$\sqrt{7}$ 无法由给定边长计算得出，可能是将 $AC$ 和 $BC$ 相减后开方。

### phys_001 串联电路电流计算

#### 📊 依赖资源

> 图片说明：包含两个电阻 $R_1=2\Omega$、$R_2=3\Omega$ 和一个电压源 $U=10V$ 的串联电路。  
> 来源上下文：第 15 页，第 7 段 | 相关文本："串联电路中，各元件电流相等..."

> 如图所示电路，已知 $R_1=2\Omega$，$R_2=3\Omega$，$U=10V$。求通过 $R_1$ 的电流 $I_1$。

| 属性 | 内容 |
| :--- | :--- |
| 题型 | 计算题 |
| 难度 | 3 |
| 分值 | 10 |
| 知识点 | `电路分析`, `欧姆定律` |

| 资源哈希 | `sha256:b1c2d3e4f5a6b7c8d9e0f1a2b3c4d5e6` |

#### 💡 解题步骤
1. 识别电路结构：$R_1$ 与 $R_2$ 串联。
2. 计算总电阻：
   $$R_{总} = R_1 + R_2 = 2 + 3 = 5\Omega$$
3. 应用欧姆定律：
   $$I = \frac{U}{R_{总}} = \frac{10}{5} = 2A$$
4. 串联电路电流相等：
   $$I_1 = I = 2A$$

答案：通过 $R_1$ 的电流为 $2A$。

## 8. 验证清单

- [ ] 文档标题格式为 `# 题库 - {YYYY-MM-DD_HHMMSS}`

- [ ] 每道题都包含完整的五个部分（标题、题干、选项、元数据、解析）
- [ ] 多模态题目的图片引用使用相对路径 `./assets/{hash}.ext`
- [ ] 元数据表格中的 `资源哈希` 字段已填写
- [ ] 所有数学公式使用 LaTeX 语法
- [ ] 判断题选项固定为 `A. 正确` 和 `B. 错误`
- [ ] 多选题的正确答案包含所有字母
- [ ] 计算题/证明题的解析包含完整步骤
- [ ] 知识点标签使用反引号包裹
- [ ] 不存在"以上都对"或"以上都不对"选项
- [ ] UI 相关题目未使用红-紫-蓝渐变色
- [ ] 题目之间使用 `---` 分隔
