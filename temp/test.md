# Jutiku Quiz Expert (出题专家)

Jutiku Quiz Expert 是一个专为从文档生成结构化试题而设计的智能体技能。它能够深入分析文档内容，识别关键知识点，并自动生成高质量的测验题目，支持多种题型和难度分级。

## ? 核心功能

- **多格式支持**：支持 PDF, DOC, DOCX 等常见文档格式（自动转换为 Markdown 处理）。
- **智能分析**：
  - **内容分类**：自动判断文档是“大纲型”还是“知识型”，采用不同的出题策略。
  - **智能分块**：针对长文档（>15k 字符）自动按章节切分，确保处理深度。
- **多样化题型**：支持单选题、多选题、判断题、填空题和简答题。
- **质量保证**：内置“自我修正循环”，进行事实核查、逻辑检查和格式验证，减少幻觉。
- **双重输出**：同时生成机器可读的 JSON 文件和便于阅读的 Markdown 试卷。

## ? 快速开始

### 前置要求
确保您的环境中已安装：
- Node.js & npm
- `markitdown` (本技能会在运行时检查并尝试安装必要的转换工具)

### 使用方法

直接在对话中通过自然语言触发：

> "根据 `产品手册.pdf` 生成 20 道题目"
> "帮我从这个文档里提取考题"
> "制作一份关于 TCP/IP 协议的测验"

## ?? 工作流程

1.  **文件准备**：定位并读取用户指定文件，非 Markdown 文件会自动转换。
2.  **策略选择**：
    -   **大纲模式**：适合结构清晰的文档，基于目录和标题扩展题目。
    -   **全覆盖模式**：适合信息密集的文档，深入段落细节出题。
3.  **题目生成**：基于分析结果生成符合难度分布（简单/中等/困难）的题目。
4.  **质量校验**：回溯原文验证答案准确性，检查选项逻辑。
5.  **结果输出**：生成的题目将保存在 `quiz/` 目录下。

## ? 输出格式

生成的试题将保存在当前目录的 `quiz/` 文件夹中，文件名为时间戳。

### 1. JSON 格式 (`.json`)
严格遵循 Schema 定义，适合导入考试系统或进行二次开发。
- **包含字段**：`qid`, `type`, `question`, `options`, `correct_answer`, `explanation`, `point`, `lev` 等。
- [查看 JSON 规范](references/QUIZ_JSON.md)

### 2. Markdown 格式 (`.md`)
格式清晰，适合人工阅读、打印或直接作为试卷使用。
- **包含板块**：元数据、题干、选项、答案、详细解析。
- [查看 Markdown 规范](references/QUIZ_MARKDOWN.md)
